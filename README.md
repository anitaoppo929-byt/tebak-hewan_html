# Tantangan Tebak Hewan

Game Tebak Hewan sederhana menggunakan HTML, CSS, dan JavaScript.

## Cara Main
1. Tebak nama hewan yang ditampilkan.
2. Setelah kedua hewan ditebak, klik **Level Berikutnya**.
3. Selesaikan semua level untuk memenangkan game.
4. Setelah selesai, klik **Main Lagi** untuk mengulang.
