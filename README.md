# Tebak Hewan

Game sederhana Tebak Hewan berbasis web (HTML, CSS, JavaScript).  
Cocok untuk belajar membuat permainan edukatif ringan.

## Cara Pakai
1. Buka `index.html` di browser.
2. Masukkan jawaban sesuai petunjuk.
3. Klik "Cek Jawaban" untuk mengetahui hasilnya.

## Struktur File
- `index.html` : Tampilan utama game.
- `style.css` : Styling game.
- `script.js` : Logika permainan.
