# Tantangan Tebak Hewan

Sebuah permainan edukatif berbasis web untuk menebak nama hewan dari gambar.

## Fitur
- 3 level permainan.
- Dua hewan per level.
- Timer 30 detik per level.
- Koin untuk membeli petunjuk (50 koin/hint).
- Koin diperoleh saat menebak hewan dengan benar.
- Layar Game Over dan Layar Selesai.

## Cara Main
1. Klik **Mulai** untuk memulai permainan.
2. Tebak nama hewan pada gambar.
3. Gunakan petunjuk jika perlu.
4. Selesaikan semua level untuk menang.
